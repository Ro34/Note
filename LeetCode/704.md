### FinalSolution
```java
class Solution {
    public int search(int[] nums, int target) {
        int big = nums.length - 1;
        int small = 0;

        while (small <= big) {
            int mid = small + (big - small) / 2;

            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                small = mid + 1;
            } else {
                big = mid - 1;
            }
        }
        return -1;

    }
}
```
